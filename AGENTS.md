# Project Instructions

## Plotting

For any plotting task in this repository, follow `PLOTTING_GUIDELINES.md`.

- Read `PROJECT_DESCRIPTION.md` before making plotting decisions, and use `Research_Work.pdf` when deeper paper context is needed.
- Use only the IEEE sizing constants from `tools/plots/plot_config.py`.
- Keep font sizes consistent across labels, ticks, legends, annotations, and colorbars.
- Save plot outputs as both PNG and PDF. Prefer `save_plot_outputs(...)` from `tools/plots/plot_config.py`.
